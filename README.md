# TinyVitals
## ht6 2026 project

## AI-Powered Contactless Baby Monitoring System

TinyVitals is an intelligent baby monitoring system that tracks vital signs, sleep patterns, and movements to help parents better understand their baby's well-being. Using AI and contactless sensing technologies, TinyVitals monitors important indicators such as heart rate, breathing, and movement without requiring any wearable devices.

## Features
- Heart rate monitoring through contactless physiological sensing
- Breathing detection and sleep pattern analysis
- Movement tracking during sleep
- AI-powered insights and alerts for unusual patterns

## How It Works
Sensors collect data about the baby's physiological signals and movements. AI models process this information to provide parents with real-time monitoring and meaningful insights through the TinyVitals dashboard.

## Technology Stack
- Python
- OpenCV
- MediaPipe
- Machine Learning Models
- mmWave Radar Sensor
- Camera-based rPPG sensing
- Real-time data processing

## Motivation
Traditional baby monitors allow parents to see and hear their baby, but they do not provide information about important health signals. TinyVitals adds an extra layer of awareness by combining AI and hardware sensing to create a safer sleep environment for infants.
